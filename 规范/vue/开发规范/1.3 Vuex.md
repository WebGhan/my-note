# Vuex

按页面建立对应的 module，一个页面对应一个 module，名称与页面名称保持一致

```
├── store
│   │── modules
│   │   ├── articles.js
│   │   └── user.js
│   └── index.js
```

## 文件

### index.js

```js
import Vue from 'vue'
import Vuex from 'vuex'

// Vuex Modules
import user from './modules/user'
import articles from './modules/articles'

Vue.use(Vuex)

export default new Vuex.Store({
  modules: {
    user,
    articles
  }
})
```
