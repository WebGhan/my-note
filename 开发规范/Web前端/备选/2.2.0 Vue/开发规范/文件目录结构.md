## 目录

```
├── build                      # 构建相关
├── mock                       # 项目mock 模拟数据
├── plop-templates             # 基本模板
├── public                     # 静态资源
│   │── favicon.ico            # favicon图标
│   └── index.html             # html模板
├── src                        # 源代码
│   ├── api                    # 所有请求
│   ├── assets                 # 字体等静态资源
│   ├── components             # 全局公用组件
│   ├── directive              # 全局指令
│   ├── filters                # 全局 filter
│   ├── icons                  # 项目所有 svg icons
│   ├── lang                   # 国际化 language
│   ├── layout                 # 全局 layout
│   ├── router                 # 路由
│   ├── store                  # 全局 store管理
│   ├── styles                 # 全局样式
│   ├── utils                  # 全局公用方法
│   ├── vendor                 # 公用vendor
│   ├── views                  # views 所有页面
│   ├── App.vue                # 入口页面
│   ├── main.js                # 入口文件 加载组件 初始化等
│   └── permission.js          # 权限管理
├── tests                      # 测试
├── .env.xxx                   # 环境变量配置
├── .eslintrc.js               # eslint 配置项
├── .babelrc                   # babel-loader 配置
├── .travis.yml                # 自动化CI配置
├── vue.config.js              # vue-cli 配置
├── postcss.config.js          # postcss 配置
└── package.json               # package.json
```


## 命名

1. 文件和页面名称始终是全部小写，横线连接 (kebab-case)  

2. 组件始终是单词大写开头，驼峰命名 (PascalCase)  

3. 全局组件始终以 App 开头，例如：AppButton.vue  

4. 和父组件紧密耦合的子组件应该以父组件名作为前缀命名，例如： ArticlesListNewsButton.vue  


## 页面

1. 每个页面对应一个文件夹  

2. 非全局的 组件 始终放在相关页面文件夹的 components 内  

3. 非全局的 js文件 始终放在相关页面文件夹的 utils 内  

4. 页面必须是独立的，不得引用别的页面文件夹内的组件  

```
├── views
│   └── articles
│       │── utils                                      # articles 所用的 utils
│       │   └── articles-handle-date.js
│       │── components                                 # articles 所用组件
│       │   ├── ArticlesButton.vue
│       │   └── ArticlesShowButton.vue
│       ├── index.vue                                  # articles 页
│       ├── articles-show
│       │   └── index.vue                              # articles-show 页
│       └── articles-list
│           │── utils                                  # articles-list 所用的 utils
│           │   └── articles-handle-date.js
│           │── components                             # articles-list 所用组件
│           │   └── ArticlesListButton.vue
│           ├── index.vue                              # articles-list 页
│           └── articles-list-news
│               │── components                         # articles-list-news 所用组件
│               │   └── ArticlesListNewsButton.vue
│               └── index.vue                          # articles-list-news 页
```
