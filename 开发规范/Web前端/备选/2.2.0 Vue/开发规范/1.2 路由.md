# 路由

按页面划分为不同的 module 文件，一个路由文件只能包含一条主路由和多个子路由。

```
├── router
│   │── modules
│   │   ├── articles.js
│   │   └── user.js
│   └── index.js
```


## 路由 Modules

### index.js 文件

```js
import Vue from 'vue'
import VueRouter from 'vue-router'

// Router Modules
import userRouter from './modules/user'
import articlesRouter from './modules/articles'

Vue.use(VueRouter)

const routes = [
  {
    path: '/',
    redirect: '/home',
    component: () => import('@/views/home')
  },
  userRouter,
  articlesRouter
]

const router = new VueRouter({
  routes
})

export default router
```


### articles.js 文件

```js
const ArticlesRouter = {
  path: 'articles',
  name: 'Articles',
  component: () => import('@/views/articles'),
  children: [
    {
      path: 'articles-show',
      name: 'ArticlesShow',
      component: () => import('@/views/articles/articles-show')
    },
    {
      path: 'articles-list',
      name: 'ArticlesList',
      component: () => import('@/views/articles/articles-list'),
      children: [
        {
          path: 'articles-list-news',
          name: 'ArticlesListNews',
          component: () => import('@/views/articles/articles-list/articles-list-news')
        }
      ]
    }
  ]
}

export default ArticlesRouter
```
